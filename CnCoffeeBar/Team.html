关于我们 ​

中文咖啡厅 是受 中文吧 启发而创建的社交群体，主旨皆在打造一个友善的 VRChat 国人交友社区。即使我们还有很长的一段路需要走，很多事情需要学习。

社区规则 ​

虽然我们无意过于干预社区，但这并不意味着你可以随意行动而不考虑后果。

  • 请勿在社区内讨论任何成员的任何相关信息，包括但不仅限于：地理位置、面容仪表、身材形态等现实内容
  • 请勿在社区内讨论包括但不仅限于：色情暴力、违法乱纪、反动言论、政治相关等容易引起反感或争辩的内容
  • 请勿在社区过度玩梗活运用抽象，尤其对其他群友的自身时间或表明兴趣的事物
  • 请勿在社区内聊骚，发情，对其他成员进行骚扰等容易造成他人困扰的行为
  • 个人不得以“中文咖啡厅”的名义进行任何公开活动

未明确列出的违规内容将依据事实情况具体判断是否适合发布，主要标准为是否让他人产生负面情绪。

对于地图内规则，将在你初次进入地图或持久化数据更新时显示。如果你忘记具体内容，请点击下方链接：

https://vrchat.com/home/world/wrld_413f924b-82d5-49e7-b4ff-0b20b5c94346/info

在打开的页面内找到 User Data 一栏，并单击 Reset User Data 按钮。之后在你进入地图时，规则将再次显示。
