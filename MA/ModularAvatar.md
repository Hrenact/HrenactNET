# Modular Avatar

Modular Avatar 是一套 **非破坏性** 工具，用于模块化你的模型和分发模型组件。通过使用 Modular Avatar，为你的模型添加新服装或模型就像搭积木一样简单！

Modular Avatar 的各项功能由单独的组件实现，所以你可以按需添加它们。它们在构建时将自动合并模型服装或动画控制器等等一系列行为……

## 如何安装

安装 Modular Avatar 的推荐方法是使用 VRChat Creator Companion。安装 VCC 后，请单击此处：

- [将 Modular Avatar 添加到 VCC](vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm.json)

然后，转到 Projects 界面，并单击 Manage Project 按钮。
在 Manage Packages 列表中找到 Modular Avatar，并单击右边的 `+` 按钮。

![AddMA](/images/MA/AddMA.png)

若要更新至最新版，单击绿色箭头 `↑` 即可自动更新。

## 预发布版本

要在 VCC 中添加预发布版本，[请单击此处](vcc://vpm/addRepo?url=https://vpm.nadena.dev/vpm-prerelease.json)

然后，前往 VCC 的 Settings 界面。取消勾选 `bd_` 并勾选 `bd_(prereleases)`，接着勾选下方的 `Show pre-release packages`

![PreMA](/images/MA/PreMA.png)

预发布版本的文档可以[在这里找到](https://modular-avatar.nadena.dev/dev)。

预发布版本通常含有大量 BUG，并且或随着开发的进展出现兼容性问题。请在 [Github 问题页面](https://github.com/bdunderscore/modular-avatar/issues) 上提供反馈和错误报告。
