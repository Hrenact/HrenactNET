# 世界固定对象 / World Fixed Object

![World Fixed Object](/images/MA/WorldFixedObject.png)

该组件可以用于使游戏对象相对于世界保持固定位置，即使玩家移动也不受影响。

## 何时应使用？

当你希望在你的模型移动时道具或物体仍然保持原位。

## 设置 World Fixed Object

将 `World Fixed Object` 组件附加到一个游戏对象上。没有可设置的配置选项。

该组件会自动在角色根部生成一个以世界原点为固定点的游戏对象，并将你的游戏对象移动到它的子对象中。你可以使用例如父约束来控制世界固定对象中游戏对象的位置。

即使使用多个世界固定对象组件，也只会生成一个约束。因此，无论使用一个还是几十个，该组件的性能影响都是相同的。

由于 Quest 的技术限制，在为 Quest 独立构建时无法使用此组件。
