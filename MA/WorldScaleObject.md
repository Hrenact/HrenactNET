# 世界缩放对象 / World Scale Object

此组件可用于强制游戏对象与世界保持相同的缩放比例，而不管当前头像的缩放比例如何。它会在游戏对象上附加一个（VRC）缩放约束，并将约束设置为相对于世界的 1,1,1 缩放比例。

## 何时应使用？

当你希望一个游戏对象随着世界缩放，而不是随着角色缩放时。这在某些复杂的约束机制中可能很有用。

## 设置 World Scale Object

只需将 `World Scale Object` 组件附加到一个游戏对象上。没有需要设置的配置选项。

请注意，`World Scale Object` 目前在 Unity 编辑器中无法预览，但在游戏中或播放模式下会正常工作。
