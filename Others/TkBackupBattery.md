# 项目介绍

这是一个简易的**外接补电模块**，可为 **HTC Tracker 3.0** 提供额外约 **20% 的续航提升**。<br>
同时，理论上它也兼容**任何使用 Type-C 接口充电的追踪器**。

在具备基础工具与材料的情况下，折算后**单个成品的制作成本约为 30 元人民币**，适合小批量自制或个人使用。

## 所需工具

- 3D 打印机（耗材推荐使用**透明 PETG**）
- 电烙铁 + 焊锡丝
- 热熔胶枪 + 胶棒
- 十字螺丝刀 **00（有时标注为 1.5）**

# 安全警告

1. **焊接锂电池相关电路时需格外小心**，请勿购买来源不明或过于廉价的锂电池。
2. 在焊接磁吸 **Pogo Pin** 之前，务必先使用磁吸充电线 **确认安装方向及正负极**。
3. 任何因复刻、使用本项目而造成的**人身伤害或财产损失，作者概不负责**。

## 组装教程

::: tip
- **焊接前测试**<br>
  在正式焊接之前，建议先对充电板进行通电测试，确认其能够正常充放电，以避免在完成组装后才发现模块故障。
- **首次使用注意事项**<br>
  在首次为补电模块充电前，请先使用模块至**不再对外供电**。<br>
  若跳过此步骤，模块可能会出现**一直显示“正在充电”**的情况。

# 使用方法

1. 将模块背面的**勾面魔术贴**，固定在绑带上**距离 Tracker 最近的毛面区域**，确保稳固不晃动。
2. 使用 **C-C 数据线/充电线**，连接补电模块与 Tracker 的 Type-C 接口。
3. 模块检测到 Tracker 后，**蓝灯常亮**，表示正在为 Tracker 供电并充电。
4. 当模块电量耗尽后，**蓝灯熄灭**。
    - 此时无需拔下数据线
    - 直接使用**磁吸充电线**连接模块
    - 即可**同时为补电模块和 Tracker 充电**

## 灯语说明

指示灯状态 | 含义
-- | --
绿灯闪烁 | 模块正在充电
绿灯常亮 | 模块已充满电
蓝灯常亮 | 模块在对外供电（为 Tracker 充电）

## 疑问解答

**Q：为什么模块充满电后放置一段时间，下次使用时只能给 Tracker 充一会就没电了？**<br>
**A：**<br>
这是正常现象。补电模块在未连接 Tracker 的情况下，**仍会持续检测外部是否存在用电需求**，该检测过程本身会产生**微小的待机功耗**。<br>
如果长时间存放未使用，模块内部电量会逐渐消耗，导致下次使用时可用电量明显减少。

**建议：**
- 尽量在使用前再为模块充电
- 避免长时间满电或空电存放
- 若需长期存放，建议每隔一段时间补一次电

**Q：为什么使用快充设备（例如酷态科）为模块充电时，指示灯不亮、无法充电**<br>
**A：**<br>
该模块**不支持任何快充协议**（如 QC / PD 等）。<br>
在使用快充充电头时，由于**协议握手阶段未检测到有效响应**，充电头可能会判定设备不兼容，从而拒绝供电，表现为指示灯不亮或无法充电。<br>

**解决方法：**
- 使用普通 5V 充电头进行充电
- 若使用充电宝，请启用**小电流 / 低功耗模式**
- 避免使用需要协议协商才能输出电压的快充设备
