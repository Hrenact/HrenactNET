# X1/P1 AMS 购买助手

![X1P1AMSHelper](/images/X1P1AMSHelper.png)

这是一个帮助以前没有 AMS，现在想为 X1/P1 系列打印机添加 AMS 设备的玩家辅助工具。

由于每种 AMS 设备附赠的配件不同，单独购买 AMS 时不一定可以开箱即用，需额外购买其它配件。

故为了最大程度的减少因采购失误导致的时间浪费，我使用 DeepSeek 编写了该 HTML 网页应用。

# 特性

- 界面直观易懂，深色模式保护眼睛
- 超出最大 AMS 总数量时弹出警告
- 可保留外挂料盘，方便打印 AMS 不兼容耗材
- 自动计算配件清单，包括购买跳转链接
- 自带自动保存，重新打开页面时无需再次设定
