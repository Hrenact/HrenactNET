# Timeline Text とは何ですか

`Timeline Text` は、歌詞専用ファイル形式 `*.lrc` を参考に開発されました。

1行の標準的な `Timeline Text` は以下の要素で構成されます：

- [mm:ss.xxx] text
- [分:秒.ミリ秒] テキスト

ここで、`text/テキスト` 部分の内容は `Module` の機能によって定義が変わります。

例えば：

- `LyricModule` では、`text/テキスト` は `Text UI` に表示する歌詞です
- `PostProcessModule` では、`text/テキスト` は `Post Process Volume` に適用する重み（ウェイト）です

## Timeline Text の動作

以下は `LyricModule` の `Timeline Text` の例です：

``` text
[00:00.000] 你好，
[00:01.000] 欢迎观看 Timeline Player System 的演示。
[00:05.000] 
[00:06.000] 现在，
[00:07.000] 你看到的是 Lyric Module 正在工作。
[00:11.000] 每一句话，
[00:12.000] 都会在正确的时间出现，
[00:14.000] 并且平滑地淡入淡出。
```

お気づきの通り、最初の歌詞を除き、歌詞を完全に表示（すなわち透明度1）するには、`Timeline Text で指定された時間 + Fade Duration × 2` の時点で完了します。

精度を重視する場合は、`Timeline Text` を作成する際に、時間を `Fade Duration × 2` だけ前倒しすることをお勧めします。
